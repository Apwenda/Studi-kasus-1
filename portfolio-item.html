Papua Sentani Explore

Platform digital untuk mengeksplorasi keindahan Danau Sentani dan budaya lokal Papua.

Proyek ini bertujuan untuk memperkenalkan keindahan alam dan budaya di sekitar Danau Sentani, Jayapura. Melalui website ini, pengunjung dapat menemukan destinasi wisata, sejarah lokal, serta galeri foto dan video.

Fitur Utama

  • Informasi lengkap tentang destinasi wisata di Danau Sentani
  • Galeri foto dan video berkualitas tinggi
  • Artikel budaya dan sejarah Papua
  • Peta interaktif untuk navigasi lokasi wisata
  • Formulir kontak dan konsultasi wisata

Teknologi yang Digunakan

  • HTML, CSS, dan JavaScript
  • Bootstrap untuk desain responsif
  • PHP dan MySQL untuk backend
  • Google Maps API untuk navigasi
